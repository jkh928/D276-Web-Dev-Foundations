# WGU D276 - Web Development Foundations

This repository contains my coursework and projects for the Western Governors University (WGU) course **D276: Web Development Foundations**.

## Course Objectives
- Build web pages using **HTML5**.
- Apply styling and layout using **CSS3**.
- Implement **Responsive Design** principles.

## Structure
- `index.html`: Main landing page for all modules.
- `Module-1-Basics/`: Intro to HTML tags and structure.
- `Module-2-CSS/`: Styling, colors, and fonts.
- `Module-3-Layout/`: Flexbox, Grid, and positioning.
